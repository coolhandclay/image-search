Clay's Image Search Abstraction Layer

Current features:
  • You can pass a query, and it will image search the query through Bing's Image API.
  • Optionally, you can limit number of results with a '?count=10' query.
  • You can see the latest searches made.
  • To return to these instructions don't include a parameter

Example usage:

https://clay-image-search.herokuapp.com/api/imagesearch/lol cats?count=1
https://clay-image-search.herokuapp.com/latest/imagesearch/
